Implement — IaC Security + SAST + Code Review
Infrastructure as Code is scanned with Checkov before deployment. Application code is scanned with Semgrep for vulnerabilities. Manual security checklist enforced via PR review gates.

❌ transaction_service.py def get_transaction(account_id): # 🚨 SQL INJECTION — user input directly # concatenated into query string! query = f"SELECT * FROM transactions " + f"WHERE account_id = '{account_id}'" return db.execute(query) def transfer_funds(req): # 🚨 No CSRF token check # 🚨 No amount validation # 🚨 Logs sensitive data amount = req['amount'] logger.info(f"Transfer: {req}") # 🚨 PII leak return db.execute("INSERT...", amount)
Fixed Code — Secure Implementation
✅ transaction_service.py def get_transaction(account_id: str): # ✅ Parameterized query — no injection query = "SELECT * FROM transactions " + "WHERE account_id = %s" return db.execute(query, (account_id,)) def transfer_funds(req: TransferRequest, token: str = Depends(verify_csrf)): # ✅ CSRF token validated by decorator amount = Decimal(req.amount).quantize( Decimal('0.01')) # ✅ Validated if amount <= 0 or amount > MAX_TRANSFER: raise HTTPException(400, "Invalid amount") # ✅ Log only non-sensitive identifiers logger.info("Transfer initiated", extra={"tx_id": req.transaction_id}) return db.execute("INSERT...", (amount,))

Key Takeaways — Why Shift-Left Security Works
  • ✓
    Threats identified before code exists — STRIDE in Phase 1 defined security requirements that drove implementation decisions
  • ✓
    IaC scanning prevents misconfiguration at scale — One Checkov rule protects every future deployment, not just this one
  • ✓
    SAST in CI makes security unavoidable — Developers get instant feedback; bugs can't merge without passing gates
  • ✓
    Sandbox + DAST found IDOR before production — A real attacker would have exfiltrated customer balances
  • ✓
    CloudTrail provides forensic accountability — Every action is auditable; the repudiation threat from STRIDE is mitigated
  • ✓
    AWS Config auto-remediation closes gaps — Infrastructure drift is detected and corrected without manual intervention
